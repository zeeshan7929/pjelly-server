Whats are the different keyword matching rules?

Comment is any
The most strict matching rule. It will match the comments only if it is similar to one of the defined keywords. If there are any additional words, letters, spaces, or symbols, the reply message will NOT be triggered. It will recognize "Hello", "HELLO" and "hello" in the same way though.

Comment begins with any
This rule requires any of the defined keywords to be placed at the beginning of the user's message so the automated reply message can be triggered.

Comment contains any
This is a more flexible matching rule. It will be triggered if any defined keyword exists any where in the user's comment. For example, if the Keyword is "morning" and the user types "Good morning, Nice to meet you." It will trigger the automated reply message.

Be-careful: if the Keyword is "like" and the user types "dislike". It will still trigger the automated reply message.

Comment contain all
This can be used for more advanced matching. It requires ALL defined keywords to exist in the user's comment. For example, if you use this matching rule with 2 keywords: "price" and "iphone", the following sentence will be matched: "What is the price of iphone X?" but this sentence will not be matched: "What is the price of mi 8?

Comment is anything
Automated reply message will be trigger for any comment including text, image or just a emoji.

How can I cancel my billing?

Login and go to "billing" then click "Cancel plan". If you cancel your subscription, you will still enjoy your plan until the end of your billing cycle.
